# dotfiles

A simple repository for managing my mac configuration

## Getting Started

### Dependencies

* M1 Macbook Pro
* MacOS Monterray

### Installing

* Checkout configuration to home directory

```
cd ~
git clone https://github.com/crblack82/dotfiles.git
```

### Executing program

* Run install.zsh from home directory
```
cd ~
./install.zsh
```
